### Nummer mit Easybell verbinden

https://www.easybell.de/hilfe/fragen/fragen-zum-telefonanschluss/antwort/sip-trunk-authentifizierung-per-fqdn-domain-einrichten/

Im easybell Kundenportal muss im Beispiel von Fonio einfach folgende Domain hinterlegt werden: `vojhwtf1hn6.eu.sip.livekit.cloud`

**Wichtig** ist, dass im Kundenportal für das eingehende Rufnummernformat **E.164 mit führendem „+"** ausgewählt wird. Nur so können wir den Anruf in diesem Format weitergeben, zum Beispiel:

`+49301234567@vojhwtf1hn6.eu.sip.livekit.cloud`

Weitere Infos dazu findest du hier:\
https://www.easybell.de/hilfe/fragen/fragen-zum-telefonanschluss/antwort/rufnummernformat-bei-eingehenden-anrufen-anpassen/

### Weiterleitungen (nur mit Cloud Anlage Pro)

Mit der Cloud Anlage Pro kann zu Nebenstellen innerhalb der Anlage weitergeleitet werden.

Szenario: ein Anrufer teilt der KI mit, dass er mit einem echten Menschen sprechen möchte. Der Anruf wird dann an eine Nebenstelle weitergeleitet, wo der Anruf angenommen werden kann.

Im Weiterleitungs Tool muss hierfür SIP ausgewählt werden. Als Ziel wird kurzwahl@easybell angegeben - z.B.: 10@easybell.
